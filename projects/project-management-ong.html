About this project

A web-based project management tool designed specifically for NGOs to help teams collaborate and track progress efficiently. Built during my internship at IRARI.

The platform provides real-time collaboration features, task management, and comprehensive reporting tools tailored to the needs of non-profit organizations.

Key Features

  • Real-time collaboration with Socket.io
  • Task and milestone tracking
  • Team management and role-based access
  • Progress reporting and analytics
  • Realtime messaging
  • Notification system for updates
