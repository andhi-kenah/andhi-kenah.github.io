ILO App

Java Android SQLite JavaScript Figma

About this project

ILO App is a mobile application designed for terminal students preparing for the tertiary technical baccalaureate exam. The app aims to encourage students and help them learn more effectively.

This personal project reflects my strategy of achieving small goals to accomplish bigger objectives, demonstrating a deep understanding of the development lifecycle and mobile technologies.

Key Features

  • Interactive quizzes and exercises
  • Progress tracking and statistics
  • Offline mode for studying without internet
  • Subject-based content organization
  • Gamification elements to boost motivation
  • Revision notes and summaries
